Description: 
This repository holds the software needed for a Internet controlled light switch.
It can function alone with a slave or be paired with a master server. 

Slave  - esp8266 flashed using arduino. 
Master - Raspberry pi running a simple LAMP server. 

Configurations possible: 
(1) Slave standalone.
(2) Web accessed light switch using LAMP server master which can be opened to the outside internet. 
(3) Master server with a tcp socket taking commands to do actions.
